Modbus Browser Help

Quick Start Guide

  1. Click "Add Server" to connect to a Modbus device
  2. Enter the server details (ID, IP address, port)
  3. Add register blocks or individual registers to monitor
  4. View real-time data in the table below

Key Features

  • Multi-Server Support: Monitor multiple Modbus devices simultaneously
  • Real-Time Updates: Live polling of register values
  • Multiple Data Formats: View values in decimal, hex, float, or boolean
  • Bulk Operations: Add multiple registers at once
  • Configuration Management: Save and load server configurations

Register Types

  • Coils (0-9999): Single-bit read/write values
  • Discrete Inputs (10000-19999): Single-bit read-only values
  • Input Registers (30000-39999): 16-bit read-only values
  • Holding Registers (40000-49999): 16-bit read/write values

Data Formats

  • Decimal: Standard numeric representation
  • Hexadecimal: Base-16 representation (0x0000-0xFFFF)
  • Float: 32-bit floating point (uses 2 registers)
  • Boolean: True/False values
Note: Float values require 2 consecutive registers. The first register contains the high word, and the second contains the low word.

Adding Registers

Single Register

  1. Click "Add Register"
  2. Select register type
  3. Enter register name and address
  4. Choose data format
  5. Click "Add Register"

Register Block

  1. Click "Add Block"
  2. Select register type
  3. Enter start address and length
  4. Click "Add Block"

Bulk Add

Use CSV format to add multiple registers at once:

name,address,format

Example:

Temperature,1000,decimal
Pressure,1001,float
Status,1002,boolean
Best Practices:
  • Start with a higher poll rate (1000-5000ms) and adjust as needed
  • Monitor only necessary registers to reduce network traffic
  • Use descriptive names for easy identification
  • Keep register blocks under 125 registers for optimal performance

Troubleshooting

  • Connection Issues: Verify IP address, port, and network connectivity
  • No Data: Check register addresses and types
  • Slow Updates: Adjust poll rate or reduce number of registers
  • Invalid Values: Verify data format matches register type

Configuration Management

  • Save Config: Click "Show Config" and then "Download Config"
  • Load Config: Click "Upload Config" and select your JSON file
Tip: Save your configuration after setting up servers and registers. This makes it easy to restore your setup later.
